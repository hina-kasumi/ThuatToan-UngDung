# Báo cáo bài tập lớn môn Phân tích và Thiết kế Thuật toán

https://algs4.cs.princeton.edu/code/

Dưới đây là danh sách các thuật toán và cấu trúc dữ liệu được triển khai dựa trên cuốn sách "Algorithms, 4th Edition"
của Robert Sedgewick và Kevin Wayne.

|  STT   | Chương  | Thuật toán / Lớp                | Mô tả                                                                                      | Trạng thái hoàn thành |
|:------:|:-------:|:--------------------------------|:-------------------------------------------------------------------------------------------|:---------------------:|
| **21** | **5.5** | `huffmanASCII`                  | Xây dựng cây Huffman cho danh sách sinh viên (không dấu).                                  |           ❌           |
| **23** | **5.5** | `LZW_ASCII`                     | Giải mã chuỗi `ABCAEOABCAABCAE` bằng thuật toán LZW.                                       |           ✅           |
| **24** | **5.5** | `LZW_UNICODE`                   | Nén và giải nén file văn bản tiếng Việt bằng LZW.                                          |           ❌           |
| **26** |    -    | `SegmentIntersection`           | Tìm các điểm giao cắt giữa các đoạn thẳng ngang và dọc bằng thuật toán Sweep-line.         |           ❌           |
| **27** |    -    | `KDTree`                        | Tìm các điểm nằm trong một hình chữ nhật và tìm điểm gần nhất bằng 2D-Tree.                |           ✅           |
| **28** |    -    | `IntervalST`                    | Cài đặt Cây khoảng (Interval Tree) để quản lý và truy vấn các khoảng 1D.                   |           ✅           |
| **29** |    -    | `RectangleIntersection`         | Tìm các cặp hình chữ nhật giao nhau.                                                       |           ❌           |
